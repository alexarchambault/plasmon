```scala
object TestModule
```

***

A module containing JVM test suites. Requires you define a [testFramework](testFramework) for your
Java, Scala or Kotlin testing library that implements the `sbt.testing` interface.
For many test frameworks, pre-configured traits are provided for you in [TestModule.Junit4](TestModule.Junit4),
[TestModule.Junit5](TestModule.Junit5), [TestModule.ScalaTest](TestModule.ScalaTest), etc.
